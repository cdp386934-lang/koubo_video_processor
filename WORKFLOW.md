## 详细步骤说明

### 步骤1: 视频转音频
- 使用MoviePy提取视频的音频轨道
- 支持多种音频格式（wav, mp3）
- 输出: `video.wav`

### 步骤2: 音频转文字（生成字幕）
- 使用Whisper或其他ASR引擎进行语音识别
- 生成带时间戳的字幕JSON
- 输出: `video.json`

**字幕格式：**
```json
{
  "subtitles": [
    {
      "Text": "恋爱到深处人会变成孩子",
      "FinalSentence": "恋爱到深处人会变成孩子",
      "StartMs": 0,
      "EndMs": 3000,
      "removed": 0
    }
  ]
}
```

### 步骤3: 去气口处理
- 检测并标记气口片段（"嗯"、"啊"、"这个"等）
- 生成去除气口后的音频文件
- 生成去除气口后的视频文件

**输出文件：**
- `video.json` - 更新removed标记
- `video_no_breath.wav` - 去气口音频
- `video_no_breath.mp4` - 去气口视频

### 步骤4: 生成关键字并贴入字幕

#### 4.1 DeepSeek AI分析字幕
- 读取字幕JSON文件
- 将字幕文本发送给DeepSeek AI
- AI分析并提取核心关键词

#### 4.2 提取核心关键词
- 每句字幕提取3-5个核心关键词
- 关键词为名词/动词，避免语气词
- 关键词简洁，不超过4个字
- 评估关键词重要性（1-3级）

**示例：**
```
输入: "恋爱到深处人会变成孩子因为爱情让人卸下防备"
输出: ["恋爱", "深处", "孩子", "爱情", "防备"]
```

#### 4.3 贴入字幕（加引号）
- 在关键词两侧添加引号标记
- 保持原文顺序和语义
- 支持多个关键词标记

**示例：**
```
原始: "恋爱到深处人会变成孩子"
贴入后: "\"恋爱\"到\"深处\"人会变成\"孩子\""
```

#### 4.4 保存到JSON
- 更新字幕JSON文件
- 添加关键词相关字段
- 保存独立的关键词文件

**更新后的字幕格式：**
```json
{
  "Text": "恋爱到深处人会变成孩子",
  "FinalSentence": "\"恋爱\"到\"深处\"人会变成\"孩子\"",
  "StartMs": 0,
  "EndMs": 3000,
  "keyword": "\"恋爱\"、\"深处\"、\"孩子\"",
  "keywords": ["恋爱", "深处", "孩子"],
  "text_grade": 3,
  "removed": 0
}
```

**字段说明：**
- `Text`: 原始字幕文本
- `FinalSentence`: 贴入关键词后的字幕（用于显示）
- `keyword`: 关键词列表（字符串格式）
- `keywords`: 关键词数组
- `text_grade`: 字幕重要性等级（1-4）

**输出文件：**
- `video.json` - 更新后的字幕文件
- `modules/output/keywords.json` - 关键词列表（JSON）
- `modules/output/keywords.txt` - 关键词报告（文本）

### 步骤5: 素材获取与插入（可选）
- 根据关键词从Pexels获取相关素材
- 在合适位置插入素材标记

### 步骤6: 添加背景音乐
- 从配置文件或模板读取音乐配置
- 添加背景音乐到草稿

### 步骤7: 应用视频信息
- 添加视频标题
- 添加作者信息
- 应用其他视频元数据

### 步骤8: 生成剪映草稿
- 根据字幕JSON和配置生成剪映草稿
- 创建draft_content.json
- 创建draft_meta_info.json

## 配置说明

### 启用关键词处理
在模板配置中设置：
```json
{
  "enable_deepseek": true,
  "deepseek": {
    "enabled": true
  }
}
```

### 配置环境变量
在 `.env` 文件中设置：
```env
DEEPSEEK_API_KEY=your_api_key
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

## 常见问题

### Q: 关键词没有生成？
A: 检查以下几点：
1. 确认 `enable_deepseek: true`
2. 确认 `.env` 中配置了 `DEEPSEEK_API_KEY`
3. 检查网络连接

### Q: 关键词没有贴入字幕？
A: 关键词会自动贴入 `FinalSentence` 字段，检查JSON文件中的该字段。

### Q: 如何查看提取的关键词？
A: 查看 `modules/output/keywords.txt` 文件，里面有详细的关键词报告。

## 技术细节

### 关键词提取算法
1. 使用DeepSeek AI的chat模型
2. 温度设置为0.3（保证稳定性）
3. 最大token数500
4. 自定义提示词优化提取质量

### 关键词贴入算法
1. 按关键词长度倒序处理（避免部分匹配）
2. 每个关键词只替换一次
3. 保持原文语义和顺序
4. 添加引号标记

### 性能优化
- 批量处理字幕
- 缓存AI响应
- 异步保存文件
- 增量更新JSON
